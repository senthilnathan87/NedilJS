About Nedil JS
Nedil JS
Nedil JS is a Simple, Lightweight, Complete and Free (Open Source) JavaScript Framework.
Nedil JS has been released under the MIT License.
  • Supports all modern browsers (IE8,9+, Firefox, Chrome, Safari, Opera,...)
  • CSS type Selectors
  • Event Handling
  • DOM Manipulation
  • Animation API
  • AJAX API
  • Data Structures
  • Common Utility API
  • Widget API
Nedil Widgets
Various types of widgets can be created using NedilJS APIs
Nedil JS has wide range of default widgets with lots of customizing options; few of those widgets are listed below
  • Button, Text Box, Check Box, Radio
  • Panel, Dialog, Tabs, Accordion
  • Calendar, Slide Show
  • Color Picker
  • Dropdown, AutoComplete, Number Roller
  • Menu, Progress Bar, Slider
  • AJAX Data Grid
  • and few more...

Nedil - Reason behind the naming
Nedil word has been taken from the Tamil (தமிழ்) language, one of the oldest languages and still spoken predominantly in southern part of India. Nedil (நெடில்) refers to long vowels which have prolonged / extended duration of utterance.

This JavaScript framework is kind of an extension to pure JavaScript with bunch of APIs and functionalities. So "Nedil" sounded like an apt name for this framework and hence it is "NedilJS" :).
